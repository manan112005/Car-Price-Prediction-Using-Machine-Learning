# Car-Price-Prediction-Using-Machine-Learning
## 🚗 Car Price Prediction Model

A Machine Learning project that predicts the estimated price of a car based on its features. This project demonstrates the complete ML workflow, from data preprocessing to building and deploying a prediction model.

### 🚀 Features

* 🧹 Data Cleaning & Preprocessing
* 🔍 Exploratory Data Analysis (EDA)
* ⚙️ Train-Test Data Splitting
* 📈 Linear Regression Model Implementation
* 🚗 Accurate Car Price Prediction
* 💾 Trained Model Saved Using Pickle

### 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Scikit-Learn
* Pickle

### 💡 How It Works

* The dataset is cleaned and preprocessed to handle missing values and prepare the data.
* Exploratory Data Analysis (EDA) is performed to understand relationships between features.
* The dataset is split into training and testing sets.
* A Linear Regression model is trained on the processed data.
* The trained model is evaluated and saved using **Pickle** for future predictions.
* Users can input car details to predict its estimated market price.

### 🎯 Project Goal

To build a reliable car price prediction model while gaining hands-on experience with data preprocessing, exploratory data analysis, linear regression, model evaluation, and machine learning model serialization using Pickle.
